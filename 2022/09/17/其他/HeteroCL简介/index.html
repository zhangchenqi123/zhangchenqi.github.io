    3.HeteroCL计算API、

    hcl.compute(shape, fcompute, name, dtype)

    • shape 定义了输出张量的形状。
    • fcompute 是一个描述元素定义的 lambda 函数。
    • name 和 dtype 是可选的。
    hcl.init()

A = hcl.placeholder((10,), "A")
B = hcl.placeholder((10,), "B")

def compute_example(A, B):
    return hcl.compute(A.shape, lambda x: A[x]+B[x], "C")

s = hcl.create_schedule([A, B], compute_example)
print(hcl.lower(s))

Out:
// attr [_top] storage_scope = "global"
allocate _top[int32 * 1]
produce _top {
  // attr [0] extern_scope = 0
  produce C {
    // attr [0] extern_scope = 0
    for (x, 0, 10) {
      C[x] = int32((int33(A[x]) + int33(B[x])))
    }
  }
}

    hcl.update(tensor, fupdate, name)

    • tensor：待更新的tensor
    • fupdate: lambda函数，描述更新动作
    • name:可选
    hcl.init()
A = hcl.placeholder((10,), "A")
B = hcl.placeholder((10,), "B")
C = hcl.placeholder((10,), "C")

def update_example(A, B, C):
    hcl.update(C, lambda x: A[x]+B[x], "U")
    
s = hcl.create_schedule([A, B, C], update_example)
print(hcl.lower(s))


Out:
// attr [_top] storage_scope = "global"
allocate _top[int32 * 1]
produce _top {
  // attr [0] extern_scope = 0
  // attr [U] storage_scope = "global"
  allocate U[int32 * 1]
  produce U {
    // attr [0] extern_scope = 0
    for (x, 0, 10) {
      C[x] = int32((int33(A[x]) + int33(B[x])))
    }
  }
}

    hcl.mutate(domain, fbody, name)

    • 该 API 允许用户使用矢量代码描述任何循环，即使循环体没有任何通用模式或包含命令式 DSL。当我们想要执行优化时，这个 API 很有用。
    • domain：描述了我们原始 for 循环的迭代域
    • fbody：for循环的主题
    • name：可选参数
    hcl.init()
A = hcl.placeholder((10,), "A")
B = hcl.placeholder((10,), "B")
C = hcl.placeholder((10,), "C")

def mut_example(A, B, C):
    def loop_body(x):
        C[x] = A[x] + B[x]
    hcl.mutate((10,), lambda x: loop_body(x), "M")

s = hcl.create_schedule([A, B, C], mut_example)
print(hcl.lower(s))


Out:
// attr [_top] storage_scope = "global"
allocate _top[int32 * 1]
produce _top {
  // attr [0] extern_scope = 0
  // attr [M] storage_scope = "global"
  allocate M[int32 * 1]
  produce M {
    // attr [0] extern_scope = 0
    for (x, 0, 10) {
      C[x] = int32((int33(A[x]) + int33(B[x])))
    }
  }
}
